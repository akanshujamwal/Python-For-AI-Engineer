### 🔹 Lecture 2 – Operators, Logic & Modules  

📚 Topics Covered:

#### ⚙️ Operators
- ➕ Arithmetic (`+ - * / % // **`)  
- 🔍 Relational (`== != > < >= <=`)  
- 🧠 Logical (`and or not`)  
- 💻 Bitwise (`& | ^ ~ << >>`)  
- 📌 Assignment (`+= -= *= ...`)  
- 🔎 Membership (`in`, `not in`)  
- 🧬 Identity (`is`, `is not`)  

#### 🔀 Control Flow
- If-Else  
- Elif Ladder  

#### 🔁 Loops
- While Loop  
- While with Else  
- For Loop  
- 🎯 Mini Project: Guessing Game  

#### 📦 Modules
- `math` → calculations  
- `keyword` → Python keywords  
- `random` → random values  
- `datetime` → date & time
